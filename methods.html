Method 1: Survey

Questions such as:
1. What do you think about the layout of the buttons on the interface?
2. What would you change about the way the controls are displayed?
3. What do you think about the location of the logo?
4. How would you change the layout to make it more simple?
5. What would you change about the controls screen?

Method 2: Heuristic Evaluation by the developer

Testing the UI to be appealing to the eye of the user, functionality in a game-sense i.e. does the UI contain all the components that it needs to get the player familiar with the game before playing.
Making sure that the layout of the buttons, text and logo is appropriate to the game and the user looking at it.
Checking for consistency and standard throughout the entire system.
Testing for minimalist design with simple interaction.
Having an option to go back to the main screen from the controls screen and vice versa.
